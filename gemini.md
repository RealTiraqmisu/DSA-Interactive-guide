# about this project
โปรเจกต์นี้คือการเตรียม Interactive Guide ให้กับ นักเรียนคณะวิศวกรรมปัญญาประดิษฐ์และการเป็นผู้ประกอบการ (หลักสูตรนานาชาติ) (หลักสูตรร่วม สจล. และ ม.กรุงเทพ), ชั้นปีที่ 2

โดยเนื้อหาคือ "จากภาษา C สู่ Python" เพราะน้องๆ เคยเขียนแต่ C แต่ยังไม่เคยเขียน Python 

เอาแบบที่พี่ดวงทำสำหรับงานอบรม DGA306 (github host: https://noswolf.github.io/DGA306/ github source code: https://github.com/noswolf/DGA306) แต่ content เป็น english ล้วนนะ

อันนี้คือวิชาที่เขาเคยผ่านมา แต่เป็นภาษาซี
พื้นฐานการเขียนโปรแกรมคอมพิวเตอร์ทางวิศวกรรม (INTRODUCTION TO ENGINEERING PROGRAMMING)
This course introduces basic concepts of computer programming such as elementary programming, data types, expressions, simple algorithms, and problem solving involving sequential statements, conditionals and iterations. Students learn routines or methods as fundamental concepts and practice using strings, arrays, lists, maps or dictionaries, pre- defined libraries and classes, abstraction mechanisms and basic object oriented programming concepts. Students will practice related activities of software development life cycle such as system requirement analysis, debugging, testing and validation.

# course guideline
## day 1
- วันแรกก็จะเน้นๆที่ Python ล่ะ ผมกะว่า จะแนะนำ course + สอน basic python programming ชม.นึง แล้ว 2 ชม.ที่เหลือคือให้นศ.ฝึกทำโจทย์ เนื้อหาควร cover พวก String, tuple, list, dict, for loop, while loop, if-else, try-catch
- แล้วช่วงบ่าย ก็จะสอน recursion ชม.นึง แล้วฝึกทำโจทย์

## day 2
